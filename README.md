# 🌿 Planika — Site Vitrine Officiel

> Bienvenue sur le dépôt du site internet officiel de **Planika** (`planika.fr`).

Ce dépôt contient le code source de la page d'atterrissage (*Landing Page*) de l'application mobile Planika. Le site est hébergé gratuitement via **GitHub Pages** et lié à notre nom de domaine chez Infomaniaak.

---

## 📱 À propos de Planika

Planika est une application mobile d'éco-bien-être conçue pour aider les utilisateurs à se connecter à leur rythme naturel à travers trois modules clés :
- 🌸 **Suivi du cycle & journal des symptômes**
- 📖 **Le Grimoire** (Cuisine saine, saine gestion des courses)
- 🌅 **Réveil progressif et naturel**

**La mécanique de gamification :** L'application intègre un compagnon végétal virtuel, le **Terramorphe**. Chaque action s'inscrivant dans une routine saine (boire de l'eau, cuisiner, bien dormir) fait grandir et évoluer le Terramorphe, transformant le soin de soi en un jeu gratifiant.

---

## 🛠️ Structure du Projet

Le site a été développé dans une optique de performance et de légèreté maximale :
- **`index.html`** : Fichier unique contenant la structure HTML5, le design adaptatif (CSS en balise `<style>`), les illustrations vectorielles des Terramorphes (SVG inline) ainsi que les micro-interactions au défilement (Vanilla JavaScript).

---

## 🚀 Déploiement

Le site est automatiquement déployé par GitHub Pages à chaque modification sur la branche principale.
- **URL de production :** `https://planika.fr`
- **Infrastructure :** Redirection DNS (Enregistrements A & CNAME) gérée depuis le manager Infomaniaak.
